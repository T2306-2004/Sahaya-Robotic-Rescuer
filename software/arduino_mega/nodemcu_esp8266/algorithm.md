  Sahaya Software Algorithm

1. Start System
2. Initialize:
   - Arduino Mega
   - ESP8266 Wi-Fi
   - ESP32-CAM
   - Microwave Sensor
   - GPS Module
3. Establish Wi-Fi connection
4. Start video streaming server
5. Loop:
   - Read microwave sensor
   - If presence detected:
       - Send alert
       - Send GPS data
   - Receive joystick commands
   - Control motors
   - Control servos
6. Monitor system status
7. Shutdown safely
8. End
